# Cyberfish App

## Development Setup

Make sure to have [Bun](https://bun.sh) and [Rust](https://www.rust-lang.org/tools/install) installed.

Install the dependencies with:

```bash
bun install
```

Then start the app in development mode with:

```bash
bun run tauri dev
```
